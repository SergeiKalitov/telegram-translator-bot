# Telegram Bot

Простой Telegram бот на Python с использованием библиотеки `python-telegram-bot`.

## Возможности

- `/start` — приветствие и список команд
- `/help` — справка
- `/echo <текст>` — повтор вашего текста
- Ответ на любое текстовое сообщение

## Установка

1. **Получите токен бота:**
   - Напишите [@BotFather](https://t.me/BotFather) в Telegram
   - Отправьте `/newbot` и следуйте инструкциям
   - Скопируйте выданный токен

2. **Установите зависимости:**

```bash
pip install -r requirements.txt
```

3. **Запустите бота:**

```bash
export TELEGRAM_BOT_TOKEN="ваш_токен_здесь"
python bot.py
```

Или одной командой:

```bash
TELEGRAM_BOT_TOKEN="ваш_токен_здесь" python bot.py
```

## Дополнительно

Для хранения токена в `.env` можно использовать библиотеку `python-dotenv` — тогда токен не нужно будет указывать в командной строке.

### Требования к системе

Для обработки голосовых сообщений на сервере или компьютере, где запущен бот, должна быть установлена утилита `ffmpeg`.
